接口和参数对照

      

        
类型端点用途主要参数

        
          
标准同步/api/v1/transcriptions文件上传后同步返回file, language, diarization, speaker_match, speaker_group, emotion, events, punctuation, hotwords, words, raw, fallback, response_format

          
OpenAI 兼容/v1/audio/transcriptions兼容 OpenAI SDKfile, model, language, prompt, response_format, timestamp_granularities，并支持增强参数

          
异步任务/api/v1/transcription-jobs长文件或远程 URL标准同步参数 + url

          
实时流式/api/v1/realtime/transcriptions浏览器麦克风实时识别session.start JSON + PCM 二进制帧

          
声纹管理/api/v1/speaker-groups注册和管理说话人group_id, name, audio

        
      

      
WebUI 只有根路径 /。不再提供 Gradio，不再使用 /ui、/web 或多页面入口。
